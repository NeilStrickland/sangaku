Sangaku poll help for teachers



The Sangaku system has a subsystem that can be used to run polls in lectures or tutorials. The intent is similar to that of Blackboard's native polling system, TurningPoint etc, but the Sangaku-based system has certain advantages.

Logging in

SoMaS staff and postgraduates can log in at https://aim.shef.ac.uk/sangaku, using their usual university user names and passwords. The system knows all the modules and tutorial groups, and which staff members are responsible for which groups. If you log in at a time when there is a session running that you are supposed to be attending, then you will automatically be taken to that session. This also works up to 10 minutes before the beginning of the tutorial. At other times, you will see a list of future sessions that you are supposed to be attending.

Students can also log in at the same URL, and the system will attempt to connect them to the right session. However, this is not totally reliable. For each Sangaku session there is a specific URL like https://aim.shef.ac.uk/sangaku/1234, which can be found as explained below; it is safer to give this to the students by email or by writing it on the board. (The SoMaS admin page where you enter tutorial attendance also has a button to send email to all members of a given tutorial group.)

The teacher's point of view

If you are a teacher and you visit the session monitor page for a given session, you will see something like this:


This shows the URL that students should use to access the session and any associated polls. You will generally want to select the Polls tab, which might look like this:

This session has only one poll. If there were more, then the first one would be shown in full and the rest would just have a title and a circled plus sign that can be used to expand.

At the top, there are buttons to select the mode. Projector mode is appropriate if the web page is being shown on a projector and so is visible to the students. Screen mode is appropriate if the web page is only visible to you. Note that you can have the page displayed in projector mode on the lectern PC and simultaneously in screen mode on your phone if you want.

At the bottom, there are buttons to control the state of the poll. Hovering over them will provide a little more explanation. By pressing them in sequence you can reveal the questions, then allow students to answer, then block further answers, then show how many people chose each option, then show which options are correct (with explanation, if one has been entered), then make everything invisible again.
